# Shirred Eggs

|      |          |
|------|----------|
| 2    | Eier     |
| 2 KL | Vollrahm |
|      | Rosmarin |

1. Ofen auf 160°C vorwärmen
3. Rosmarin + 2 KL Vollrahm pro Geschirr
4. Je Ein Ei ins Geschirr schlagen
5. Mit Salz + Pfeffer (+ Paprika) würzen
6. 14-16min Backen
7. Wenn Eiweiss fast hart ist und Eigelb noch weich rausnehmen
